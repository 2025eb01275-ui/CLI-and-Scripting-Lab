# Question 1 - Explanation

## Objective
The objective of this task is to create a shell script that copies student submission files to a backup folder and generates a report.

## Command 1
```bash
mkdir -p ~/CLI-and-Scripting-Lab/Question1
```
**Explanation:** Creates the Question1 directory. The `-p` option creates parent directories if they do not already exist.

## Command 2
```bash
cd ~/CLI-and-Scripting-Lab/Question1
```
**Explanation:** Changes the current working directory to the Question1 folder.

## Command 3
```bash
pwd
```
**Explanation:** Displays the current working directory to verify the correct location.

## Command 4
```bash
nano duplicate_backup.sh
```
**Explanation:** Opens the Nano text editor to create and edit the shell script.

## Command 5
```bash
chmod +x duplicate_backup.sh
```
**Explanation:** Gives execute permission to the shell script.

## Command 6
```bash
mkdir submissions
```
**Explanation:** Creates a folder named `submissions` to store sample files.

## Command 7
```bash
echo "Student A" > submissions/file1.txt
```
**Explanation:** Creates `file1.txt` and writes "Student A" into it.

## Command 8
```bash
echo "Student B" > submissions/file2.txt
```
**Explanation:** Creates `file2.txt` and writes "Student B" into it.

## Command 9
```bash
cp submissions/file1.txt submissions/file1_copy.txt
```
**Explanation:** Creates a duplicate copy of `file1.txt`.

## Command 10
```bash
./duplicate_backup.sh
```
**Explanation:** Executes the shell script to create the backup and generate the report.

## Command 11
```bash
ls
```
**Explanation:** Lists all files and folders in the current directory.

## Command 12
```bash
cat report.txt
```
**Explanation:** Displays the contents of the generated report file.

## Command 13
```bash
cat errors.txt
```
**Explanation:** Displays any error messages generated by the script.

## Command 14
```bash
ls backup
```
**Explanation:** Lists the files present in the backup folder.

## Result
The shell script executed successfully. The backup folder was created, files were copied successfully, the report was generated, and no errors were found.
