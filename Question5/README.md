## Recovery Mechanisms in vi Editor

This folder contains the solution for Question 5 of the CLI and Scripting Lab.

### Contents

### Summary

This question evaluates the recovery mechanisms available in the vi editor, including swap files, undo history, registers, backup files, and auto-recovery. It concludes that swap files with auto-recovery provide the most reliable method for recovering unsaved work after a system crash.
